# Quick Start Guide

## Get Started in 3 Steps

### Step 1: Install Dependencies

```bash
cd supply_chain_platform
pip install -r requirements.txt
```

### Step 2: Run the Application

```bash
streamlit run app.py
```

### Step 3: Explore Features

The application will open in your browser at `http://localhost:8501`

## What You'll See

### Home Page
- Overview of supply chain vulnerability
- Key metrics and statistics
- Quick navigation to all features

### 5 Main Features

1. **🎮 Shock Simulator**
   - Visualize disruption timeline
   - Industry and GDP impacts
   - Recovery projections

2. **🗺️ Risk Map**
   - Interactive global map
   - Regional risk comparison
   - Critical dependencies

3. **🛒 Consumer Impact**
   - Product shortages
   - Price increases
   - Spending changes

4. **💼 Investment Intelligence**
   - Buy opportunities
   - High-risk sectors
   - Portfolio strategy

5. **📄 Executive Summary**
   - Comprehensive report
   - Key findings
   - Strategic recommendations
   - Downloadable report

## Customization

Use the **sidebar** to:
- Select different shock scenarios
- Adjust intensity (10-100%)
- Navigate between features

## Default Scenario

**Taiwan Semiconductor Disruption at 40% Intensity**

This represents a moderate supply chain shock affecting:
- 15 countries globally
- 12 major industries
- 15 consumer product categories

## Tips

- 📊 All charts are interactive - hover for details
- 🔄 Change parameters in sidebar to see real-time updates
- 📥 Download executive summary as text file
- 🖱️ Click on map markers for country details

## Example Workflow

1. Start on **Home** page - review overall impact
2. Go to **Shock Simulator** - understand timeline
3. Check **Risk Map** - see geographic distribution
4. Review **Consumer Impact** - product-level analysis
5. Explore **Investment Intel** - opportunities and risks
6. Generate **Executive Summary** - comprehensive report
